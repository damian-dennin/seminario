# Stackr

Red social para conectar personas con proyectos tech colaborativos. Sistema de emparejamiento estilo swipe: deslizá para mostrar interés en proyectos, generá matches con los dueños y coordiná por chat.

## Requisitos

- Python 3.x
- Cuenta en [Supabase](https://supabase.com)

## Instalación

```bash
pip install -r requirements.txt
```

## Ejecución

```bash
python app.py
```

Abrí `http://localhost:5000` en el browser.

## Stack

- **Backend:** Flask (Python)
- **Base de datos:** Supabase (PostgreSQL)
- **Frontend:** HTML / CSS / JavaScript vanilla
